# Web Image Downloader

Утилита командной строки для массового скачивания изображений с веб-страниц.

## Возможности

- Скачивание всех изображений со страницы
- Поддержка относительных и абсолютных ссылок
- Распознавание форматов: JPG, PNG, GIF, WebP, SVG, BMP
- Детальная статистика загрузки
- JSON отчет о результатах
- Оптимизированная загрузка с задержками


## Установка

```bash
# Клонирование репозитория
git clone https://github.com/ТВОЙ_НИК/web-image-downloader.git
cd web-image-downloader

# Установка зависимостей
pip install -r requirements.txt
